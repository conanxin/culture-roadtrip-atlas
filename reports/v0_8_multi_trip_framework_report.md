## 主要变更

### 1. 首页升级
- 项目定位说明：为自驾、徒步、古建、博物馆与历史地理旅行准备的个人导游手册库
- 顶部导航：推荐 / 路线库 / 方法论 / 关于 / GitHub
- Hero 区：路线库 / Featured Trip / 规划中 / 已上线 标签

### 2. Featured Trip 区块
突出展示「北京出发·辽塔巡礼自驾导游手册」作为当前主推路线，含状态、主题、核心标签、"进入行程"按钮。

### 3. 路线库
6 条路线卡片，每张含：
- 路线名称、短名、一句话说明
- 状态：已上线 / 规划中
- 天数、起终点、核心主题
- 适合人群、标签、当前进度
- 按钮：已上线显示"进入行程"，规划中显示"规划中"

### 4. 筛选功能
8 个分类筛选按钮：全部 / 古建 / 石窟 / 博物馆 / 自驾 / 徒步 / 边疆史 / 城市史。使用原生 JS 实现，无框架。

### 5. 路线状态说明
- 已上线：已有完整页面，可直接使用
- 规划中：已有主题方向，等待整理
- 待实地复盘：路线已规划，等待实际走访后修订
- 已实地复盘：已根据真实旅行更新

### 6. 项目方法论模块
"一条路线如何被整理" 8 步框架：
1. 路线骨架
2. 景点档案
3. 随行导游词
4. 城市与美食
5. 行前书影音
6. 资料来源核对
7. 现场模式
8. 实地复盘

### 7. 数据与渲染
- 新增 assets/js/trips-data.js：路线库数据
- 新增 assets/js/home.js：首页渲染/筛选逻辑
- 现有辽塔行程页保持不变
