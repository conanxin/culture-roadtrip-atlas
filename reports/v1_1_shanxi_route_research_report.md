## 主要变更

### 1. 山西页升级：v0.9 → v1.1
页面状态从「规划中骨架页」升级为「研究建档版」。所有具体信息保留为「待核对/待校准」。

### 2. 候选点位分级
- **A 核心必看（12 个）**：云冈石窟、华严寺、善化寺、应县木塔、佛光寺、南禅寺、晋祠、镇国寺、双林寺、小西天、玉皇庙、青莲寺
- **B 强烈推荐（5 个）**：双塔寺、介休后土庙、崇庆寺、法兴寺、长治观音堂
- **C 可选补充（2 个）**：平遥古城、云冈石窟周边博物馆

每个点位卡片增加：
- 路线角色
- 路线段（1-5）
- 状态标签（开放时间待核对 / 门票待核对 / 停车待核对）
- 资料可信度

### 3. 5 段路线研究框架
| 段 | 主题 | 重点 |
|----|------|------|
| 1 | 北京 → 大同 | 云冈石窟、华严寺、善化寺（北魏、辽金寺院） |
| 2 | 大同 → 应县 → 五台山周边 | 应县木塔、佛光寺、南禅寺（唐构、辽塔） |
| 3 | 太原 → 晋中 | 晋祠、镇国寺、双林寺（宋、五代、明） |
| 4 | 临汾 / 隰县方向 | 小西天（明悬塑） |
| 5 | 长治 / 晋城 / 晋东南 | 崇庆寺、法兴寺、玉皇庙、青莲寺（宋金元密集区） |

### 4. 资料来源索引（6 类）
- 官方文旅资料（山西省文旅厅、各市文旅局）
- 文物 / 博物馆资料（国保单位名录、研究院、博物馆）
- 建筑史资料（梁思成著作、柴泽俊著作等）
- 纪录片 / 视频（CCTV 系列）
- 书籍 / 论文（《文物》《考古》等期刊）
- 旅行整理说明（社区攻略 + 本作者研究笔记）

### 5. 山西古建关键词（10 个）
木构建筑 / 斗栱 / 梁架 / 彩塑 / 壁画 / 密檐塔 / 辽金寺院 / 晋东南古建 / 地方信仰 / 佛教空间

每个词条包含：
- 一句话解释
- 为什么和本路线有关
- 对应点位

- **8 天压缩版**：北京 → 大同 → 应县/五台 → 太原 → 平遥/介休 → 临汾/隰县 → 长治/晋城 → 北京
