行旅图谱 是一个个人人文旅行项目库。每条路线都按统一框架整理：路线骨架、景点档案、随行导游词、城市与美食、行前书影音、资料来源核对、现场模式、实地复盘。

与旅游攻略不同，行旅图谱追求的不是"打卡清单"，而是随行导游式的沉浸体验——每到一个地方，你会知道为什么这个地方值得来，它在中国（乃至世界）历史和文化的坐标中处于什么位置，应该怎么看、怎么感受、怎么思考。

技术层面，这是一个纯静态网站：HTML + CSS + Vanilla JavaScript，部署于 GitHub Pages。在浏览器原生能力的范围内（localStorage、speechSynthesis）实现尽可能丰富的交互。

关于行旅图谱

行旅图谱 是一个个人人文旅行项目库，收录自驾、徒步、古建、博物馆、历史地理主题路线。每条路线均包含完整的行程规划、景点深度讲解、随行导游词、城市背景、美食推荐与行前资料包。

与旅游攻略不同，行旅图谱追求的不是"打卡清单"，而是"随行导游"式的沉浸体验。每到一个地方，你会知道为什么这个地方值得来，它在中国历史和文化的坐标中处于什么位置，应该怎么看、怎么感受、怎么思考。

设计风格

视觉风格参考文物图录与古代长卷：墨绿、米白、灰绿、暗金的色调，长卷式的排版布局，克制的装饰语言，古朴而不喧嚣的整体气质。每一次阅读，都像是展开一卷泛黄的史料。

内容原则

  • 内容深度优先：重点景点提供 800 字左右的随行导游词
  • 原创讲解：基于历史文献和实地考察，不是资料的搬运
  • 多层次设计：重点景点有完整导游词，普通景点有简明介绍
  • 可持续扩展：架构设计支持未来添加更多路线
  • 资料可追溯：每条信息标注来源与可信度
